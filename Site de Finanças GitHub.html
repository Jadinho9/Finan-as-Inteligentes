Para rodar esse site no GitHub Pages, siga este passo a passo detalhado:

📁 1. Crie os arquivos no seu computador

Crie uma pasta e dentro dela salve dois arquivos:

Arquivo index.html

Copie o conteúdo que criei anteriormente (já estilizado e com estrutura completa) e salve com o nome index.html.

Arquivo styles.css

Copie o conteúdo do CSS e salve como styles.css na mesma pasta.

🌐 2. Crie um repositório no GitHub

  1. Vá até github.com e faça login.

  2. Clique em "New repository".

  3. Dê um nome como: financas-inteligentes.

  4. Marque como público.

  5. Clique em "Create repository".

⬆️ 3. Envie os arquivos para o repositório

Você pode fazer isso de duas formas:

🔹 Usando a interface web:

  1. No repositório, clique em "Add file" > "Upload files".

  2. Selecione os arquivos index.html e styles.css.

  3. Clique em "Commit changes".

🔹 Ou usando Git local (opcional):

bash
git init
git remote add origin https://github.com/SEU_USUARIO/financas-inteligentes.git
git add .
git commit -m "Primeiro commit"
git push -u origin master

🚀 4. Ative o GitHub Pages

  1. No repositório, vá em Settings > Pages.

  2. Em Branch, escolha main e selecione a pasta / (root).

  3. Clique em Save.

Você verá o link gerado:

arduino
https://SEU_USUARIO.github.io/financas-inteligentes/

### Como salvar e publicar seu site no GitHub Pages

#### 1. Acesse os arquivos
Você tem dois arquivos criados aqui:
- `index.html` (estrutura do site)
- `styles.css` (estilo do site)

#### 2. Baixe os arquivos
Clique nos três pontinhos (⋯) no canto do arquivo aberto e selecione **“Export as file”** para baixar cada um deles.
Salve ambos no mesmo diretório do seu computador.

#### 3. Crie um repositório no GitHub
1. Vá para [https://github.com](https://github.com) e faça login.
2. Clique em **“New repository”**.
3. Nomeie como `financas-inteligentes` (ou como desejar).
4. Marque como **público** e clique em **Create repository**.

#### 4. Envie os arquivos
1. Dentro do repositório, clique em **Add file > Upload files**.
2. Envie `index.html` e `styles.css`.
3. Clique em **Commit changes**.

#### 5. Ative o GitHub Pages
1. Vá em **Settings > Pages**.
2. Em **Branch**, selecione `main` e pasta `/root`.
3. Clique em **Save**.
4. Pronto! O GitHub fornecerá o link do seu site publicado.
